Platforms

  • Live BLE works in Chrome or Edge on Windows, macOS, Linux, or Android. You must open this app from http://localhost or HTTPS.
  • iPhone / iPad (live BLE): use the native SwiftUI app in this repo at ios/HELPStatCompanion (Xcode + CoreBluetooth). This web UI on iOS can only import JSON for viewing unless Web Bluetooth is added by Apple in the future.
  • Install on home screen: use your browser’s “Add to Home Screen” for a full-screen shortcut; offline use is limited to viewing imported data once cached.
